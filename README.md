# Expense Tracker

Minimal Next.js skeleton for the Expense Tracker application.

## Repository

This repository contains the starter code for the Expense Tracker project.

Live dev server (local): `http://localhost:3000`

## Quick Start

1. Install Node.js (v18+ recommended) and npm.
2. From the project root run:

```bash
npm install
npm run dev
```

3. Copy `.env.example` to `.env.local` and add required environment variables for Supabase, Clerk, etc.

## Project Structure

- `pages/` – Next.js page components.
- `styles/` – global styles.
- `public/` – static assets.
